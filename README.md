osgi-ddthcollection-commons

## Installation #

Latest release version: `0.1.1`. See [RELEASE-NOTES.md](RELEASE-NOTES.md).

Maven dependency:

```xml
<dependency>
	<groupId>com.github.ddth</groupId>
	<artifactId>ddth-osgicommons</artifactId>
	<version>0.1.1</version>
</dependency>
```
